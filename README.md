#### 一款简洁美观的 Android 音乐播放应用，让你のlife充满旋律。


## 📱 应用介绍

**LifeMusic** 是为 Android 平台开发的本地音乐播放器，主打简洁 UI 与流畅体验。无广告、无打扰，只为让你专注于音乐本身。

---

## ✨ 核心功能

- 🎧 本地音乐扫描与播放
- 📁 歌单管理（创建 / 编辑 / 删除）
- 🔁 支持顺序播放、单曲循环、随机播放
- 🎼 显示音乐信息（标题、专辑、艺术家）
- 🌙 深色 / 浅色主题切换
- 🔊 支持后台播放与通知栏控制
- 拥有 musicfx / equalizer（第三方）
- support lyrics font size
